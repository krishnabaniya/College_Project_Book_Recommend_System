ℹ️ About This Project

This web-based Book Recommendation System helps users explore trending, popular, and personalized books using hybrid filtering methods, including collaborative and content-based algorithms. Users can rate, review, subscribe, and discover books based on their interests and genre preferences.

  • 📌 Genre-Based Search
  • 📌 User Reviews and Ratings
  • 📌 Real-time Suggestions
